Introduction

What You'll Learn

In this tutorial, you will build a simple LED blinker which blinks the USR LED on and off at 1 Hz. By the end, you'll know how to create a basic RTL design, map signals to physical pins, use the open-source FPGA toolchain, and program the board.

What You'll Need

  • Bluejay FPGA development board
  • FTDI-based programmer (either a FTDI C232HM USB-to-serial cable or an Adafruit FT232H breakout board)

Tutorial Structure

The tutorial is broken into four parts, listed below. Each page builds on the previous one, so we recommend following them in order.

  1. Installing the Tools - Install the open-source FPGA toolchain
  2. Creating Your First Design - Write the RTL code and define the pin constraints
  3. Building the Bitstream - Generate the bitstream using the open-source toolchain
  4. Programming the Board - Load your design onto the FPGA

Next Steps
