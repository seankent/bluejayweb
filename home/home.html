Overview

The Bluejay is a low-power FPGA development board built around the iCE40LP8K from Lattice Semiconductor.

The Bluejay provides a convenient, flexible platform for working with the iCE40LP8K. The board maximizes the logic and I/O resources available to the designer while keeping onboard features to a minimum. Additional functionality, if needed, can be connected through the board's 39 programmable I/O pins. This approach allows designers to include only what's required for their specific application, keeping designs lean and power-efficient.

Key Features

  • iCE40LP8K FPGA: The largest FPGA in the iCE40 LP/HX lineup.
    • 7,680 Look-Up Tables (LUTs)
    • 128 kbits of Embedded Block RAM (EBR)
    • 2 Phase Locked Loops (PLLs)
  • 39 Programmable I/O Pins: Maximum flexibility for interfacing with sensors, displays, and other devices
  • Onboard Configuration Storage: 2 Mbit SPI flash for persistent bitstream storage
  • Flexible Power Options: Power via USB-C, 5V pin, or 3.3V pins to suit your application
  • Dual Configuration Modes: Master Mode for standalone operation, Slave Mode for rapid prototyping
  • Ultra-Low Power: Only 600 ÂµA idle consumption, ideal for battery-powered applications
  • Open-Source Toolchain: - Complete RTL-to-bitstream workflow using free and open-source tools.

Getting Started

Ready to start developing? Check out our Getting Started Guide to install the toolchain, build your first design, and program the board.

New to FPGAs? The Bluejay is a great platform for learning digital design. Start with the simple LED blinker example and work your way up to more complex projects. View Getting Started Guide.

Open-Source Toolchain

The Bluejay works seamlessly with a completely free and open-source FPGA development toolchain:

Yosys - Verilog/SystemVerilog synthesis NextPNR - Place and route for iCE40 devices Project IceStorm - Bitstream generation and programming tools

No expensive proprietary software licenses required. The complete toolchain runs on Linux, macOS, and Windows (via WSL), making FPGA development accessible to everyone. See the Software Installation guide to get started.
