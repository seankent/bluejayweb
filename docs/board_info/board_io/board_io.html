Board I/O

Header Pins

The Bluejay provides access to 39 of the iCE40's Programmable I/O (PIO) pins. These pins are exposed through two 24-pin headers, J1 and J2, and are labeled D0–D38. Each PIO can be configured as an input, output, or tri-state buffer.

In addition to the PIO pins, the board includes a configuration interface for programming the iCE40 and flash chip. This interface consists of an SPI bus (CS, SCK, MOSI, and MISO) and two dedicated iCE40 configuration pins, CRST and CDN.

The J1 and J2 headers also include several 3V3 and GND pins, as well as a 5V pin. Depending on the power delivery method, these pins can act as either power inputs or outputs.

Aside from the 5V power pin, all of the board's I/O operates at 3.3V logic levels.
