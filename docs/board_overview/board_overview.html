Board Overview

Overview

The Bluejay is a low-power FPGA development board designed around the Lattice iCE40LP8K. The board provides direct access to 39 of the FPGA's programmable I/O pins, along with essential supporting components for power delivery, clock generation, and configuration storage.

Key features include a 12 MHz oscillator, 2 Mbit configuration flash, flexible power input options (USB-C, 5V, or 3.3V), and dual configuration modes (Master and Slave). The board's minimalist design keeps power consumption low—approximately 2 mW when idle—making it well-suited for battery-powered applications.

The board works with completely free and open-source development tools, providing a complete workflow from RTL design to bitstream programming.

Callouts

iCE40LP8K FPGA

The centerpiece of the Bluejay is the iCE40LP8K, an ultra-low-power FPGA from Lattice Semiconductor. Key features include:

  • 7,680 Look-Up Tables (LUTs)
  • 128 Kbits of Embedded Block RAM (EBR)
  • 2 Phase-Locked Loops (PLLs)

39 Programmable I/O Pins

The board provides access to 39 of the FPGA's programmable I/O pins through two 24-pin connectors (J1 and J2). Each pin can be independently configured as a digital input, output, or tri-state buffer. This high pin count and programmability allows the FPGA to interface with a wide variety of external devices, such as sensors, displays, microcontrollers, or other FPGAs.

12 MHz Oscillator

An onboard 12 MHz oscillator provides a clock signal for the system. The oscillator output can be used directly as a clock source or as a reference clock for the FPGA's internal PLLs to generate higher or lower frequencies as needed.

Onboard Configuration Storage

The board includes a 2 Mbit SPI flash chip for storing the FPGA's configuration bitstream. This allows the bitstream to be stored onboard and loaded automatically when power is applied.

The flash can be programmed through the board's CONFIG interface.

Status LEDs

The board includes two status LEDs for visual feedback. The PWR LED indicates the board is receiving power. The USR LED is connected to one of the FPGA's programmable I/O pins and is available for application use.

When operating in power-constrained applications, the status LEDs can be disconnected using the LED EN jumper.

Multiple Configuration Modes

The board supports both Master and Slave configuration modes. In Master Mode (MM), the FPGA automatically loads its configuration from the flash when power is applied. In Slave Mode (SM), an external programmer connects to the board and loads the configuration directly into the FPGA.

The configuration mode is selected using a jumper.

Flexible Power Options

The board can be powered through the USB-C connector, the 5V pin, or the 3V3 pins.

When 5V is supplied to the board—either through the USB-C connector or the 5V power pin—an onboard low-dropout regulator (LDO) steps down the voltage to 3.3V. In this configuration, the 3V3 pins can be used as power outputs to supply external circuitry (up to the LDO's current limit).

When 3.3V is supplied directly to the 3V3 pins, the LDO is bypassed. This allows the board to be easily integrated into larger systems that already have a regulated 3.3V supply.

Low Power Design

The Bluejay is designed for low-power applications, consuming just 2 mW when idle. Active power consumption varies based on FPGA resource utilization and switching frequency. When clocked at the nominal 12 MHz frequency, the board typically consumes anywhere from a few milliwatts to the low tens of milliwatts. Even when the clock is boosted to its maximum frequency of 275 MHz (using one of the FPGA's internal PLLs) and switching activity is high, power consumption typically remains well under 1 watt.

Open-Source Toolchain

The Bluejay works seamlessly with completely free and open-source software that runs on Linux, macOS, and Windows. This includes a complete RTL-to-bitstream toolchain as well as tools for programming the FPGA and onboard flash.

Notable tools include:

  • Yosys - RTL synthesis
  • NextPNR - Place and route
  • Project IceStorm - Bitstream generation and programming utilities
