Installing the Tools

Overview

To develop for the Bluejay, you'll need to install a set of tools to convert your RTL design into a bitstream and load that bitstream onto the FPGA. The good news is that the entire toolchain is completely free and open-source.

Required Tools

The process of generating and loading a bitstream requires four main tools:

  • Yosys - Synthesizes Verilog/SystemVerilog code into a netlist
  • NextPNR-ice40 - Maps the netlist to physical FPGA resources (place-and-route)
  • IcePack - Converts the placed design into a binary bitstream
  • IceProg - Programs the bitstream onto the iCE40 FPGA via an FTDI-based programmer

Together, these tools combine to form a complete RTL-to-bitstream toolchain.

Installation

Using OSS CAD Suite (Recommended)

The OSS CAD Suite from YosysHQ includes all the required tools pre-built for Linux, macOS, and Windows.

Download the suite:

Visit the OSS CAD Suite releases page and download the appropriate package for your operating system.

Extract and configure:

Once downloaded, extract the archive to a location of your choice (e.g., /opt/oss-cad-suite or ~/oss-cad-suite).

Add the tools to your path:

Terminal
export PATH="<extracted_location>/oss-cad-suite/bin:$PATH"

To make the tools available in every new terminal session, add the export command to your shell configuration file (.bashrc, .zshrc, etc.).

Verifying the Installation

To confirm that all tools are correctly installed and accessible, run the following commands:

On Linux/macOS:

Terminal
which yosys
which nextpnr-ice40
which icepack
which iceprog

On Windows:

Terminal
where yosys
where nextpnr-ice40
where icepack
where iceprog

Each command should return a path to the respective tool.

Additional Tools

The OSS CAD Suite includes numerous other open-source tools useful for Verilog and FPGA development. Notable among these are:

  • Icarus Verilog (iverilog) - Verilog simulator for testing your designs
  • GTKWave - Waveform viewer for analyzing simulation results and debugging
  • Cocotb - Python-based framework for writing testbenches

Although these tools are not required for this tutorial, we highly recommend exploring them as you develop more complex designs.

Next Steps

With your tools installed, you're ready to create your first FPGA design!
